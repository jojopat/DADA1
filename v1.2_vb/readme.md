Legacy source code: DADA v1.2 (VB.NET)
"The foundational algorithms that paved the way for modern RF analysis."
This directory contains the original source code for the Directional antenna deployment assistant (DADA) version 1.2. This version represents a critical era in the project's 45-year evolution, transitioning the core logic from Foxbase/Clipper origins into the VB.NET framework.

Why this version matters
While the project has now moved to a modern HTML/JS architecture (v3.3), this legacy VB.NET code holds the original implementation of:

The core RF calculation engine: The initial translation of complex propagation formulas into executable code.

Historical context: A look into how DADA functioned before becoming a browser-based tool.

Status
Development era: Developed during the transition to .NET before the current web version.

Platform: Windows / .NET Framework.

Maintenance: This version is now Archived. For active development and the latest features, please refer to the root directory of this repository.

Open source & public domain
Like all versions of DADA, this legacy code is released into the Public Domain under The Unlicense. It is shared freely for educational purposes and historical reference for the amateur radio community.
